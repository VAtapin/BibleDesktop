# 1. Введение

## 1.1 Название проекта

Bible Desktop 2.0

---

## 1.2 Основная идея проекта

Bible Desktop — это не программа для чтения Библии и не очередной Bible Reader.

Bible Desktop — это современная многоязычная платформа для глубокого изучения Библии, объединяющая:

* Web Application
* REST API
* Telegram Bot
* Telegram Mini App
* Orthodox Calendar
* AI Bible Assistant
* Semantic Search
* Strong Numbers
* Cross References
* Notes & Bookmarks
* Group Bible Study

---

## 1.3 Главная цель

Создать современную платформу изучения Библии, которая позволит:

* читать Священное Писание;
* изучать его индивидуально;
* работать в группах;
* использовать ИИ для анализа текста;
* выполнять смысловой поиск;
* работать с номерами Стронга;
* использовать православный календарь;
* использовать Telegram как полноценный интерфейс работы.

# 11. Bible Reader

## Основная задача

Bible Reader является центральной частью системы.

Пользователь должен иметь возможность быстро открыть любой перевод, книгу, главу или стих.

---

## Функции Reader

Поддерживаются:

```text
выбор перевода
выбор книги
выбор главы
выбор стиха
выбор диапазона стихов
переход по ссылке
история переходов
открытие нескольких вкладок
```

---

## Вкладки

Система должна поддерживать несколько открытых вкладок.

Пример:

```text
Бытие 1
Псалом 90
Матфея 5
Римлянам 8
```

Каждая вкладка хранит собственное состояние.

---

## Режимы отображения

```text
один перевод
два перевода рядом
несколько переводов
параллельное чтение
```

---

## Контекстное меню стиха

При выборе стиха доступны действия:

```text
копировать текст

создать заметку
мой перевод
толкование

открыть Стронга
открыть параллельные места
```

# Telegram Bot Platform

## Назначение

Telegram Bot является одной из основных частей Bible Desktop.

Bot должен быть полностью самостоятельным продуктом и выполнять большинство задач без необходимости открытия сайта или Telegram Mini App.

Mini App используется только для сложных сценариев работы.

Основная задача бота:

* ежедневное взаимодействие с пользователем;
* чтение Библии;
* православный календарь;
* поиск;
* AI-помощник;
* планы чтения;
* работа с группами;
* уведомления.
